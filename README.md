# My Dictionary App
My Dictionary is a simple web-based dictionary application created using HTML, CSS, and JavaScript.  
It allows users to search for English words and view their meanings, examples, synonyms, and antonyms using a dictionary API.

This project was built to practice JavaScript API handling and DOM manipulation.

## Features
- Search English words
- Displays meaning and part of speech
- Shows example usage
- Lists synonyms and antonyms
- Clean and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Public Dictionary API

## Project Purpose
The main goal of this project was to understand:
- How to fetch data from an API
- How to work with JSON responses
- How to dynamically update content on a webpage using JavaScript

## How to Use
1. Enter a word in the search box
2. Click on the search button
3. The meaning and related details will be displayed
